# CDLHOMINGPIGEON

中文名称： 家鸽

用法：预示着趋势反转

调用方法：signal = CDLHOMINGPIGEON(open, high, low, close)

输出：0代表不符合k线形态，100表示符合形态。

## 指标介绍

二日 K 线模式，与母子线类似，不同的的是二日 K 线颜色相同，第二日最高价、最低价都在第一日实体之内
